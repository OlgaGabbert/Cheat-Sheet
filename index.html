    3. Element Usage
    <table> Creates table
    <tr> Adds rows to a table before adding table data and table headings
    <thead> Defines the headings of table columns encapsulated in table rows
    <th> Adds titles to rows and columns of a table and must be enclosed in a table row element
    <tbody> Contains all table data other than table heading and table footer content. Table cannot have both this element and table row as direct children.
    <td> Inside a table row element, adds a cell of data to a table
    <tfoot> Uses table rows to give footer content, to summarize content at the end of a table.
